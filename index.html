Keto Meals

Sneak Peek
Live Site
Source code

Keto meals is a database website for keto food created with flask. The primary objective of the project was to allow users to create, update, delete and view recipes. Users are also able to 'favourite' recipes from other users in order to access them easily through their profile page.
The secondary purpose of the site was to promote a brand of cookware, these promotions are seen throughout the site under recipes and the products advertised are automatically created to match the tools required for each recipe.

Technology used:

HTML, CSS, PYTHON, FLASK, MONGODB

Henna Styles

Sneak Peek
Live Site
Source code

Henna Styles is an ecommerce site for henna and henna accessories, built using the django framework and stripe payment system. Visitors can browse products in a variety of ways, add to their shopping basket and checkout as well as accessing the community page where they can view posts and comments made by other henna enthusiasts. Registered Users can view their past orders, save their personal information and add to the community section.

Technology used:

HTML, CSS, PYTHON, DJANGO, SQL, STRIPE
